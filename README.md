### 1. Описание системы

**АИС (Автоматизированная информационная система) кафедры** предназначена для хранения информации по вопросам, интересующим студентов и преподавателей кафедры, оперативного доступа к этой информации, а также удобного ввода и обновления данных.

### 2. Требования к базе данных

База данных должна содержать следующую информацию:

*   **О преподавательском составе:** ФИО, должность, читаемые курсы, научные интересы, расписание пребывания.
*   **Об учебной работе кафедры:** читаемые курсы, программы дисциплин, методические материалы.
*   **О направлениях научной работы:** состав направлений, их краткая характеристика, участники научных разработок.
*   **О структуре кафедры:** секции, руководители секций, заведующий кафедрой и его заместители.

#### Технические требования:

*   Предусмотреть хранение изображений (минимум в одном поле).
*   Реализовать хранимые процедуры и/или триггеры.
*   Обеспечить наличие не менее 50 записей в базе данных.

### 3. Технологический стек

*   **СУБД:** `PostgreSQL`
*   **Язык программирования:** `С#`

### 4. Структура проекта

Репозиторий содержит исходный код приложения, а также расчетно-пояснительную записку со следующей структурой:

1.  **Анализ технического задания** ✅
2.  **Разработка моделей данных** (концептуальная, логическая, физическая)
3.  **Разработка и реализация АИС**
4.  **Тестирование АИС**
5.  **Приложения:**
    *   Приложение 1. Модели данных
    *   Приложение 2. Текст программы
    *   Приложение 3. Снимки окон программы (скриншоты)
